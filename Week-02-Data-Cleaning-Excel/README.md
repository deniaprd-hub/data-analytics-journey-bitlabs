# 🧹 Week 2: Data Cleaning & Exploratory Data Analysis (Excel)

## 📌 Deskripsi
Minggu ini berfokus pada proses penyiapan data (*Data Preparation*) menggunakan Microsoft Excel/Spreadsheet. Saya mempraktekkan alur kerja pembersihan data untuk memastikan analisis yang dihasilkan akurat dan valid.

## 🛠️ Langkah Pembersihan Data
- **Handling Duplicates:** Mengidentifikasi dan menghapus data ganda pada dataset e-commerce.
- **Handling Missing Values:** Menangani data kosong agar tidak merusak hasil perhitungan.
- **Data Transformation:** Mengubah format data (seperti tanggal dan angka) agar sesuai untuk diolah.
- **Outlier Detection:** Mencari anomali data, seperti usia karyawan yang tidak legal (ditemukan karyawan usia 16 tahun).

## 📈 Teknik Analisis
- **Pivot Tables:** Digunakan untuk merangkum data karyawan, seperti mencari masa kerja terlama (39 tahun) dan distribusi gender.
- **Formula Excel:** Menggunakan rumus perbandingan dan logika untuk validasi data operasional.
