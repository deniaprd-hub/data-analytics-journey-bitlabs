## 🛠️ Kemampuan Python (Library Pandas)
Saya mempraktikkan pengolahan dataset menggunakan Python di Google Colab:
- **Data Filtering:** Mengambil data spesifik, seperti pelanggan dengan metode pembayaran *Electronic Check*.
- **Sorting Data:** Mengurutkan transaksi berdasarkan biaya bulanan (*Monthly Charges*) tertinggi secara otomatis.
- **Data Locating:** Menggunakan fungsi `.iloc` dan `.loc` untuk mengambil baris data tertentu dengan presisi.
- **Statistika Deskriptif:** Menghasilkan ringkasan data (Mean, Median, Max) hanya dengan satu baris kode `.describe()`.
