# 🗄️ Week 4: SQL Fundamentals (SQLite)

## 📌 Deskripsi
Pada minggu ini, saya mempelajari dasar-dasar bahasa kueri SQL menggunakan SQLite. Fokus utamanya adalah memahami bagaimana database relasional bekerja dan cara mengambil informasi spesifik dari beberapa tabel sekaligus.

## 🛠️ Keterampilan SQL yang Dipraktikkan
- **DDL (Data Definition Language):** Membuat struktur tabel baru (seperti tabel `karyawan`) dengan *Primary Key* dan tipe data yang sesuai.
- **DML (Data Manipulation Language):** Menambahkan data baru (*INSERT*), memperbarui informasi (*UPDATE*), dan menghapus data (*DELETE*).
- **Data Querying:** Mengambil data dengan filter tertentu menggunakan `WHERE` dan `HAVING`.
- **Table Joins:** Menggabungkan tabel `Customer`, `Invoice`, dan `InvoiceLine` untuk melihat total pembelian per pelanggan.

## 🔍 Contoh Kasus
Saya berhasil membuat kueri untuk menampilkan pelanggan yang memiliki total pembelian lebih dari 5 item dengan menggabungkan tiga tabel berbeda menggunakan perintah `JOIN` dan fungsi agregasi `SUM`.
