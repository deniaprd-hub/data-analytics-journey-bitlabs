# 🚀 Week 1: Innovation Process & Design Thinking

## 📌 Deskripsi
Pada minggu pertama, saya mempelajari bagaimana mengidentifikasi masalah bisnis menggunakan kerangka kerja **Design Thinking** dan merancang solusi inovatif untuk meningkatkan pengalaman pelanggan.

## 🧠 Konsep Utama
- **User Journey Map:** Memetakan langkah-langkah pelanggan (Deni) dalam mencari tempat kursus data analis.
- **Innovation Type:** Fokus pada **Profit Innovation** (Model Bisnis).
- **BPMN (Business Process Model and Notation):** Merancang alur proses bisnis dari browsing produk hingga pesanan terkirim.

## 📊 Hasil Analisis (Key Insights)
- **Problem:** Pelanggan (karyawan/mahasiswa) membutuhkan fleksibilitas waktu dalam belajar.
- **Solusi Inovasi:** Pengadaan **Mini Course Gratis** sebagai strategi pemasaran untuk memberikan *experience* awal kepada calon pelanggan sebelum membeli kelas utama.
- **BPMN Flow:** Berhasil merancang alur transaksi e-commerce yang mencakup pengecekan stok otomatis dan notifikasi pengiriman.
